# ST2195 Assignment 3: Airline Database Construction

This repository contains scripts to construct an SQLite database from the **Data Expo 2009** airline dataset and perform relational queries.

### Data Sources
* **Data Expo 2009:** [Link to Data](https://stat-computing.org/dataexpo/2009/)
* **Harvard Dataverse:** [Link to Harvard Dataverse](https://dataverse.harvard.edu/)

### Contents
* `r_sql/`: R scripts using `DBI` and `dplyr`.
* `python_sql/`: Python scripts using `sqlite3` and `pandas`.
